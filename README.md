# Mini-Projet : Catalogue de Produits Interactif (L3 Génie Informatique)

## Objectif du Projet
L'objectif est de concevoir une application web simple et moderne permettant de gérer et de visualiser un catalogue de produits. Les étudiants doivent choisir entre deux frameworks populaires : **React** ou **Vue.js**.

Ce projet permet de mettre en pratique les concepts fondamentaux du développement frontend :
- Création de composants réutilisables.
- Gestion de l'état local (State).
- Passage de données via les propriétés (Props).
- Manipulation des événements.
- Filtrage de données dynamiques.

---

## Fonctionnalités Attendues
1. **Affichage du Catalogue** : Présenter une liste de produits avec image, nom, prix et catégorie.
2. **Recherche & Filtrage** : Permettre à l'utilisateur de rechercher un produit par nom ou de filtrer par catégorie (ex: Électronique, Vêtements, etc.).
3. **Panier d'Achat (Simulation)** : Ajouter un bouton "Ajouter au panier" qui met à jour un compteur global.
4. **Interface Responsive** : L'application doit être présentable sur ordinateur et mobile (utilisation de CSS simple ou Flexbox/Grid).

---

## Structure du Dépôt
- `starters/` : Contient les fichiers de démarrage pour React et Vue.
- `glossaire.md` : Guide des termes techniques essentiels pour comprendre le framework choisi.
- `README.md` : Ce fichier.

---

## Instructions pour les Étudiants
1. Choisissez votre camp : **React** ou **Vue**.
2. Allez dans le dossier `starters/` correspondant.
3. Installez les dépendances (`npm install`).
4. Lancez le serveur de développement (`npm start` ou `npm run dev`).
5. Commencez l'implémentation dans le fichier principal (`App.js` ou `App.vue`).
